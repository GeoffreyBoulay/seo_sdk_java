Release 1.1.1 - 31 July 2013

  • 1.1.1 standalone JAR - bvseosdk-1.1.1.jar
  • 1.1.1 JAR with dependencies - bvseosdk-1.1.1-jar-with-dependencies.jar
  • release notes
    • Related to the 1.1 configuration changes, additional default properties are now included in the JAR so clients don't need to include bvclient.properties in order to define all default properties.

Release 1.1 - 26 July 2013

  • 1.1 standalone JAR - bvseosdk-1.1.jar
  • 1.1 JAR with dependencies - bvseosdk-1.1-jar-with-dependencies.jar
  • release notes
    • SDK support for multiple configuration instances added
    • a default Configuration object is still built from the properties file defaults if defined
    • Note: the method signature for the static getBVContent() methods has now changed. They require a Configuration object to be passed in.

Release 1.0.1 - 26 June 2013

  • 1.0.1 standalone JAR - bvseosdk-1.0.1.jar
  • 1.0.1 JAR with dependencies - bvseosdk-1.0.1-jar-with-dependencies.jar
  • release notes
    • Adds support for loading bvclient.properties from anywhere in the classpath
    • Any user agent string containing "Google" now includes the SEO content

Release 1.0

  • 1.0 standalone JAR - bvseosdk-1.0.jar
  • 1.0 JAR with dependencies - bvseosdk-1.0-jar-with-dependencies.jar
